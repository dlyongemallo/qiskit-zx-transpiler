# Benchmarking

Scripts for measuring `ZXPass` against Qiskit's own optimiser on the
[QASMBench](https://github.com/pnnl/QASMBench) suite.

## Running locally

```bash
cd benchmarking
git clone https://github.com/dlyongemallo/QASMBench.git  # if not already cloned
./run_benchmarks.sh
```

`run_benchmarks.sh` invokes `run_benchmarks.py` and tees the output to
`benchmarks_output.txt`.

## `benchmarks_output.txt` is a versioned performance snapshot

The file is checked in deliberately. CI (`.github/workflows/test.yml`) copies
it to `benchmarks_output.bak`, re-runs the benchmark, and `diff`s the two to
flag any regression or unexpected change from a PR. When a PR legitimately
shifts the numbers (e.g., a new optimisation pass), update the snapshot in the
same PR so the diff captures the intended effect.

The plot images (`Depth_compression_ratio.png`,
`Ratio_of_non-local_gates.png`) are regenerated by `run_benchmarks.py` and
committed alongside the text snapshot for the same reason.
